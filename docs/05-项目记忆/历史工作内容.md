# 历史工作内容

> 项目：家庭记账桌面应用（family-accounting-app）
> 开发周期：2026-07-04 ~ 2026-07-05
> 技术栈：Tauri 2 + React 18 + TypeScript + MUI 5 + AG Grid 32 + Rust + SQLite

---

## 10 步增量开发总览

### Step 1 ✅ 项目脚手架 + 数据库 + 布局
- Tauri 2 + Vite + React + TypeScript + MUI + Tailwind 初始化
- SQLite 10 张表 + 15 个索引
- 初始数据：24 个消费标签 + 24 条汇总映射 + 122 条分类规则 + 5 条应用设置
- 侧边栏导航（9 个菜单项）+ 路由配置

### Step 2 ✅ 账单导入 + 归属人选择
- 微信 XLSX 解析器（calamine，跳过 17 行头部，1899-12-30 纪元修正闰年 bug）
- 支付宝 CSV+GBK 解析器（encoding_rs + csv，跳过 22 行头部）
- SHA-256 文件哈希去重
- 归属人 CRUD（account_owners 表）
- 导入时选择归属人（支持跳过）

### Step 3 ✅ 数据清洗 + 明细合并
- 过滤状态：还款成功、交易关闭、退款成功、不计收入、已全额退款
- 金额 ≤ 3 元过滤（仅 expense 方向）
- 部分退款处理：regex 提取退款金额，≤ 3 元忽略、> 3 元重算
- 预览 → 勾选确认 → 执行（先修改后排除）
- 交易明细页面：月份筛选 + 汇总统计 + 已排除行灰色删除线

### Step 4 ✅ 分类规则引擎 + 人工复核
- 三层匹配引擎：AI 学习规则（confidence DESC）→ 内置/用户规则（priority ASC）→ 留空
- 三种匹配方式：exact（精确）、like（包含）、in（列表）
- 匹配字段优先级：counterparty > product > transaction_type
- 人工复核页面：自动分类按钮 + 未分类列表 + 单条/批量指定标签
- 交易明细页面标签内联编辑（乐观更新 + 失败回滚）

### Step 5 ✅ 月度汇总 + 下钻跳转
- 按月份 + 消费标签 GROUP BY + SUM
- 通过 summary_mappings 合并子标签到 16 个汇总类
- 汇总下钻：点击金额 → 跳转 `/transactions?month=&tags=` 自动筛选
- 多月汇总透视表（行=月份，列=汇总类+手动数据）
- 悬浮显示子项构成

### Step 6 ✅ 手动数据录入 + AG Grid
- 月度手动数据录入表单（总资产/收入/房贷/理财/保险/分析）
- 支持动态子项分解（如总资产 = 中行 ¥200k + 支付宝 ¥500k）
- AG Grid 替换 MUI Table：行内编辑、键盘导航、复制粘贴、排序/筛选/列拖拽
- 批量操作栏：批量标签/归属人/刚需/删除
- 新增交易条目 + 批量新增

### Step 7 ✅ AI 辅助分类（DeepSeek）
- DeepSeek API 模块（provider trait + deepseek 实现 + prompts）
- AI 分类：未匹配交易 → DeepSeek 批量判断 → ai_suggested_tag + ai_confidence
- 人工复核页面三步流程：① 自动分类 → ② AI 分类 → ③ 人工复核
- 采纳/修正/跳过 + AI 学习规则生成
- 已跳过交易管理（查看+恢复）
- 设置页面：API Key 配置 + 测试连接

### Step 8 ✅ AI 月度分析 + 报表生成
- AI 月度分析按钮 → DeepSeek 分析当月消费 → 保存到 ai_reports 表
- AI HTML 报表生成 → 安全过滤（去除 script/iframe/on*）→ WebView 渲染
- 报表历史管理（查看/全屏）
- AI 图表生成（输入自然语言描述 → DeepSeek 生成 Recharts 配置 → 渲染图表）

### Step 9 ✅ 规则管理 UI + 标签管理 + 映射管理 + 规则测试
- 分类规则管理：新增/编辑/删除/启停（exact/like/in）
- AI 学习规则管理：完整交易上下文展示 + 置信度 + 确认/修正次数
- 消费标签管理：新增/编辑/删除（系统内置不可删）
- 汇总映射管理：配置汇总类与标签映射
- 规则测试：输入字段值预览匹配结果
- 归属人管理合并到规则管理页面

### Step 10 ✅ 增强与优化
- 导出 CSV、批量删除、批量采纳
- 右键菜单（标记刚需/排除/添加规则/复制单元格）
- 列宽持久化到 localStorage
- AG Grid 中文 locale + 筛选重置按钮
- 归属人下拉筛选器（自定义 floating filter）
- 标签 Chip 着色（24 种颜色）
- 交易时间格式化 + 来源列移除 + 序号列
