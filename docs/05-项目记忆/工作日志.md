## 系统架构最终态

### 后端（Rust）
```
src-tauri/src/
├── ai/           — DeepSeek 适配器 + Prompt 模板
├── commands/     — 52 个 Tauri 命令
├── dao/          — 8 个数据访问层文件
├── db/           — SQLite 连接 + 迁移
├── models/       — 9 个数据模型
├── parsers/      — 微信 XLSX + 支付宝 CSV 解析器
└── services/     — 5 个业务逻辑服务
```

### 前端（React）
```
src/
├── api/          — 9 个 API 封装文件
├── components/   — Layout/Sidebar + ai/reports/transactions
├── pages/        — 8 个页面组件
├── types/        — TypeScript 类型定义
├── utils/        — 导航工具
└── styles/       — AG Grid 自定义样式
```

### 数据库
10 张表 + 52 个命令 + 16 个汇总类 + 24 个标签 + 122 条内置规则
