# 家庭记账桌面应用 — 使用说明

## 快速开始

### 1. 配置归属人
进入「规则管理」→「归属人」Tab → 添加家庭成员（如 joey、vila）

### 2. 导入账单
- 进入「导入账单」→ 选择微信 XLSX 或支付宝 CSV 文件
- 选择账单归属人（或跳过）→ 确认导入
- 支持多月账单文件

### 3. 数据清洗
- 进入「数据清洗」→ 点击「开始清洗」
- 查看待过滤（还款/退款/小额）和待修改（部分退款）列表
- 可逐条取消勾选 → 确认执行

### 4. 自动分类
- 进入「人工复核」→ 点击「① 自动分类」运行规则引擎
- 点击「② AI 分类」由 DeepSeek 辅助判断（需先在设置中配置 API Key）
- 逐条复核：确认 / 跳过 / 采纳 AI 建议 / 修改标签
- 选中多行可批量操作

### 5. 查看和编辑交易
- 「交易明细」页面使用类 Excel 表格（AG Grid）
- 双击单元格编辑标签/归属人/刚需/商品/金额
- 选中多行出现批量操作栏（标签/归属人/刚需/删除）
- 右键菜单：标记刚需/排除、添加规则、复制
- 支持 Ctrl+C/V 复制粘贴、排序、筛选、列拖拽
- 点击「新增」按钮手动添加交易或批量新增

### 6. 月度汇总
- 进入「月度汇总」查看全部月份收支透视表
- 每行 = 一个月，列 = 各汇总类金额
- 点击分类金额可下钻到该月该分类的交易明细
- 悬浮表头或数据单元格查看子项构成
- 底部填写月度手动数据（总资产/收入/房贷/理财/保险/分析）
- 每个字段支持动态添加子项明细（如总资产 = 中行 + 支付宝 + ...）

### 7. AI 功能（可选）
- 进入「设置」→ 填入 DeepSeek API Key → 启用 → 保存 → 测试连接
- 人工复核页面使用 AI 辅助分类
- 报表中心使用 AI 月度分析和 HTML 报表生成
- 报表中心支持自然语言描述生成图表

### 8. 报表中心
- 进入「报表中心」→ 选择月份
- 「AI 分析」：DeepSeek 分析当月消费
- 「生成 HTML 报表」：DeepSeek 生成美观报表页面（支持全屏查看）
- 「图表生成」：输入描述（如"各月餐饮支出趋势"）→ AI 生成图表
- 底部查看历史报表记录

### 9. 规则管理
- 进入「规则管理」管理：
  - 分类规则（新增/编辑/删除/启停，支持精确/包含/列表匹配）
  - AI 学习规则（查看学习记录和置信度）
  - 消费标签（新增/编辑/删除，系统内置不可删）
  - 汇总映射（配置汇总类与标签的映射关系）
  - 规则测试（输入字段值预览匹配结果）
  - 归属人（新增/编辑/删除）

### 10. 数据导出
- 交易明细页面工具栏点击「导出」→ 下载 CSV 文件
